# 📋 帳單工具使用說明

這個工具幫助你在 Google Sheets 中**一鍵生成格式化的帳單**，方便寄給客人。

---

## 🎯 工作流程一覽

```
   客人下單
       ↓
   訂單進到 Google Sheets
       ↓
   你確認庫存、計算運費
       ↓
   ★ 點「📋 帳單工具 → 產生帳單預覽」
       ↓
   輸入運費 + 截止日
       ↓
   跳出帳單視窗
       ↓
   點「📋 一鍵複製全文」
       ↓
   去 Gmail 貼上 → 寄給客人
```

---

## 🔧 首次設定（只需做一次）

### Step 1：更新 Apps Script

如果你之前已經部署過舊版的 Apps Script，請更新成新版：

1. 打開你的 Google Sheets
2. 上方選單 **擴充功能 → Apps Script**
3. 把 `Code.gs` 的所有內容刪除
4. 開啟 `website/GoogleAppsScript.gs`，把全部內容複製貼上
5. **確認第 9 行的 `SPREADSHEET_ID` 已填入你的試算表 ID**
6. 點💾 儲存
7. 點 **部署 → 管理部署作業 → ✏️編輯 → 版本選新版本 → 部署**

### Step 2：授權新權限

新版多了「讀取試算表並彈出對話框」的權限，因此第一次執行需要重新授權：

1. 回到你的 Google Sheets
2. **重新整理頁面** (F5)
3. 你會在上方選單看到新的「📋 帳單工具」選單出現
4. 點擊 **📋 帳單工具 → 一鍵設定（首次使用）**
5. 第一次會跳出授權視窗，請點「繼續」並允許所有權限
6. 完成後會看到「✅ 設定完成！新增 X 個欄位。」

### Step 3：確認試算表新增了 3 個欄位

「訂單總覽」工作表會自動新增 3 欄：

| 欄位 | 說明 |
|------|------|
| **運費(元)** | 你確認運費後填入的金額（也可在帳單對話框中填，會自動回填） |
| **繳款截止日** | 你設定的繳款期限（也可在對話框中填） |
| **狀態** | 訂單目前狀態（下拉選單） |

「狀態」欄會自動有下拉選單：`待確認 / 已發帳單 / 已匯款 / 已出貨 / 取消`

## 📨 日常使用 - 處理一筆訂單

### 1. 進到「訂單總覽」工作表

點擊你要處理的那筆訂單的**任何一個儲存格**（例如點訂單編號那格）

### 2. 點選單 「📋 帳單工具 → 產生帳單預覽（選中那筆訂單）」

### 3. 跳出視窗詢問運費

- 如果客人選**自取**，預設 0
- 如果客人選**宅配**，預設 130
- 確認箱數後輸入實際金額（例如客人買了 2 箱就填 260）
- 點「OK」

> 💡 之後再執行同一筆訂單時，會直接讀取已填的金額，不再詢問。

### 4. 跳出視窗詢問繳款截止日

- 預設今天 + 5 天
- 也可以自己輸入（例如 `2026/05/20`）
- 點「OK」

### 5. 跳出「帳單預覽」視窗

視窗中會看到兩個版本的帳單：

#### 📧 純文字版（推薦給一般客人）

完整的純文字帳單，包含商品明細、金額、匯款資訊、截止日、注意事項。

**點「📋 一鍵複製全文」** → 到 Gmail 開新郵件 → 貼上 → 寄出

#### 📄 HTML 格式版（給講究排版的客人）

帶有顏色、表格、強調金額的精美版本。

**點「複製為 HTML」** → 到 Gmail 開新郵件 → 直接貼上（保留所有樣式）→ 寄出

### 6. 狀態自動更新

完成後試算表中該筆訂單的「狀態」欄會自動變成「**已發帳單**」。

## 📝 帳單會包含什麼

每張帳單都會自動包含：

- **訂單資訊**：訂單編號、訂購人、公司、電話、取貨方式、地址
- **商品明細**：每樣商品的名稱、規格、未稅單價、數量、小計
- **金額計算**：
  - 商品未稅金額
  - 營業稅 5%
  - 商品合計
  - 運費
  - **★ 應付總金額**（醒目顯示）
- **匯款資訊**：銀行、戶名、帳號（已預先填好你的資訊）
- **繳款截止日**（黃色警示框）
- **客人下單時填的備註**（如果有）
- **注意事項**：要求客人提供匯款末 5 碼等

## 💡 進階使用

### 修改後重新產生帳單

如果客人臨時要修改數量或加減商品：

1. 在「**訂單明細**」工作表手動修改數量或刪除整列
2. 在「**訂單總覽**」工作表同步修改「未稅金額」「營業稅 5%」「應付總金額」這 3 欄
   - 或用公式：未稅金額 = SUM 對應明細小計
3. 再次執行「產生帳單預覽」即可

### 缺貨處理

如果某商品缺貨：

1. 到「**訂單明細**」找到該筆訂單對應的那一列商品，**整列刪除**
2. 到「**訂單總覽**」更新該筆訂單的金額欄（手動扣掉那項商品的金額與稅金）
3. 在「備註」欄補一行：「⚠ 商品 OO 缺貨已扣除」
4. 重新產生帳單，客人就會看到調整後的明細

### 設定預設運費

如果你想簡化流程、不要每次都問運費，可以直接在「**運費(元)**」欄手動填上金額（執行帳單預覽時就會跳過詢問）。

### 自訂繳款截止日

可以提前在「**繳款截止日**」欄填入日期，產生帳單時就會跳過詢問。

## ❓ 常見問題

### Q1：選單「📋 帳單工具」沒出現？

- **重新整理試算表**（F5）後再看一次
- 確認 Apps Script 部署已更新且有授權

### Q2：點選了訂單但說「請點選一筆訂單列後再執行」？

- 確認你點的是**有資料的那一列**（不是第一列標題列、也不是空白列）
- 確認你在「**訂單總覽**」工作表上，不是「訂單明細」

### Q3：執行時跳出權限要求？

第一次執行 `onOpen` 或 `setupSheet` 之外的功能時，需要額外授權「彈出對話框」「呼叫 UI」的權限。請允許後再執行。

### Q4：要怎麼把舊資料補上「運費」「截止日」「狀態」欄位？

執行一次「📋 帳單工具 → 一鍵設定（首次使用）」就會自動補欄位。重複執行不會破壞已有資料。

### Q5：Gmail 貼上後格式跑掉？

- 如果用「純文字版」：請確認 Gmail 你是在「**純文字模式**」貼上（右下角 `⋮` 可切換）
- 如果用「HTML 版」：請確認 Gmail 是「**多重格式模式**」（預設就是）

### Q6：可以批次處理多筆訂單嗎？

目前一次處理一筆。但對 1-10 筆/週的量來說，每筆 30 秒就能完成，總共也不到 5 分鐘。

### Q7：帳單寄出後客人匯款了，怎麼更新狀態？

在「訂單總覽」找到該筆訂單，「狀態」欄下拉選單選「**已匯款**」即可。出貨後再改成「**已出貨**」。

## 🎬 一個完整的範例

假設你今天收到 3 筆訂單，要寄帳單給他們：

1. **早上 9 點**：打開 Google Sheets 看到 3 筆新訂單
2. **9:05**：第一筆 - 點訂單列 → 帳單工具 → 輸入運費 130 → 截止日 5/20 → 複製 → Gmail 寄出。狀態自動變「已發帳單」
3. **9:08**：第二筆 - 客人自取，運費 0 → 同上操作
4. **9:11**：第三筆 - 客人買 2 箱要宅配，運費 260 → 同上操作
5. **三筆都完成，約花 10 分鐘**

之後就等客人匯款，匯款進來就把狀態改為「已匯款」→ 出貨後改「已出貨」，整個流程結束。
